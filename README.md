# Bulk Document Automation - Excel VBA

https://github.com/Tanisha-analysis/Bulk-Document-Automation-Excel-VBA/blob/main/InShot_20260602_132931858.mp4

**Impact**: Reduced manual PDF creation from 2+ hours to 5 minutes for 500+ records | 96% time saved

## Key Features
- Excel ↔ Word COM automation
- Dynamic placeholder replacement in body text + textboxes
- Production error handling + logging to sheet
- Auto PDF naming + illegal character filtering

Excel VBA, Word COM, FileSystemObject

Built in IST (UTC+5:30) for async collaboration with US/EU teams.
